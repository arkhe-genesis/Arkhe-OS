## Starting the System

### Quick Start
To launch the system core backend:
cd arkhe-os
pip install fastapi uvicorn
uvicorn main:app --port 8000 &

To view the Nexus Dashboard:
cd arkhe-os/ui/desktop
npm install
npm run dev &

The system will connect and verify coherence. Welcome to the Cathedral!
