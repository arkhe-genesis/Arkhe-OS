# Substrate 304 Extensions

Includes Federated Learning for phi_c weight optimization, real-time dashboard visualization, and 50+ inter-species node expansion.

To run the dashboard:
`streamlit run optimization/dashboards/phi_c_dashboard.py`

To use the federated learning optimizer:
See `substrate_304_extensions/federated_learning.py`

To run the 50+ node inter-species expansion:
`python3 substrate_304_extensions/inter_species_diversity.py`
