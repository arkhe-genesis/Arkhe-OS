# Prove ROI on AI Search

## Quick start

To start tracking ROI on AI Search, you must establish baselines for emerging generative KPIs using advanced LLM tracking tools.

## Workflows

1. **Define Generative KPIs**
   - Identify target metrics: Citation Traffic, Brand Mention Frequency, and Zero-Click dominance.
   - Establish current baseline values for each metric.
2. **Track Metrics**
   - Configure advanced LLM tracking tools to monitor the defined KPIs over time.
   - Set up automated data collection pipelines.
3. **Report on ROI**
   - Synthesize the tracked data into actionable insights.
   - Generate reports demonstrating how the generative KPIs move the needle on overall ROI.

## Advanced features

- Use automated dashboards to visualize Citation Traffic and Brand Mention Frequency in real-time.
