# Pesquisa: Reconhecimento de Emoções em Português

Esta pesquisa explora o uso de *Weak Supervision* para o reconhecimento de emoções no idioma português, utilizando como base técnica o repositório `cewebbr/PortugueseEmotionRecognitionWeakSupervision`.

## 🔬 Relevância para a Catedral
O reconhecimento preciso de emoções é fundamental para o pilar de **Integridade Epistêmica**. Ao entender a carga emocional de um dado, o sistema pode:
1. Detectar manipulações sutis (Sentiment Bias).
2. Ajustar a coerência de fase (λ) do **NeuralScaffold** com base no feedback afetivo do usuário.
3. Refinar a resposta dos **Esquadrões Cognitivos** para manter a neutralidade e a verdade.

## 🛠️ Detalhes Técnicos (Weak Supervision)
A abordagem de *Weak Supervision* permite treinar modelos robustos sem a necessidade de conjuntos de dados massivos e rotulados manualmente, o que é ideal para nichos do português (brasileiro vs. europeu).

- **Fonte**: [GitHub Repository](https://github.com/cewebbr/PortugueseEmotionRecognitionWeakSupervision)
- **Métodos**: Snorkel, Labelling Functions, Modelos Heurísticos.
