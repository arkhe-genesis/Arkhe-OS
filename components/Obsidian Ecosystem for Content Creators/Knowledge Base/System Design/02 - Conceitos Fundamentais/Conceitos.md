# 02 - Conceitos Fundamentais

Antes de projetar sistemas complexos como a **Catedral**, precisamos dominar as bases.

## 🌐 Networking Basics
- **HTTP/HTTPS**: O padrão da web. Na Catedral, evoluímos para o **qHTTP**.
- **DNS**: Resolução de nomes. O `Arkhe-DNS` estende isso para mapear conceitos em endereços IPv8.

## 🏗️ Arquitetura Cliente-Servidor
O modelo básico onde um cliente solicita recursos e o servidor os fornece. Na Arkhe-Network, isso é descentralizado via P2P.

## ⚡ Latência vs Throughput
- **Latência**: Tempo para uma mensagem ir da origem ao destino (Ex: ms em `NetworkInfraState`).
- **Throughput**: Quantidade de dados processados em um tempo dado (Ex: Envelopes/sec em `TzinorNetwork`).
