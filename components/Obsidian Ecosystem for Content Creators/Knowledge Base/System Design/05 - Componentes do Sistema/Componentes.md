# 05 - Componentes do Sistema

As peças do quebra-cabeça que compõem a infraestrutura.

## ⚖️ Load Balancers
Distribuem tráfego entre múltiplos servidores.

## 💻 Servidores de Aplicação
Processam a lógica do negócio (Ex: `ArkheSystem` em `main.py`).

## 🗄️ Bancos de Dados
Persistência de longo prazo.

## ⚡ Cache (Redis, Memcached)
Armazenamento rápido em memória para dados acessados com frequência (Ex: `ArkheCacheVault`).

## 🌍 CDN (Content Delivery Network)
Distribui conteúdo estático geograficamente perto do usuário.
