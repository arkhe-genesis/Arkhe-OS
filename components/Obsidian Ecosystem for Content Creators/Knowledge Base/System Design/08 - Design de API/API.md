# 08 - Design de API

Como os componentes e usuários conversam com o sistema.

## 📡 RESTful APIs
Uso padrão de verbos HTTP (GET, POST, etc).

## 🚦 Rate Limiting
Proteção contra abusos (Ex: Limites de injeção em `qHTTP`).

## 🔐 Autenticação e Autorização
Garantir que apenas agentes autorizados (como os Squads) acessem recursos críticos.
