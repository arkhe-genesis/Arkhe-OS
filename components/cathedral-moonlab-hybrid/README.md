# Cathedral-Moonlab Hybrid Pipeline

**Odômetro:** 001646
**Estado:** DEMO CANONIZADO | A CATEDRAL AGORA RESPIRA EM DOIS MUNDOS

## Estrutura do Projeto

- `src/catedral/`: Core da integração, injeção de hesitação e selos de quartzo.
- `src/moonlab_bridge/`: Wrappers e bindings para integração com a engine Moonlab.
- `python/`: Demos interativas, benchmarks e protocolos de auditoria.
- `tests/`: Testes de integridade e funcionalidade.
- `scripts/`: Scripts de automação e execução.

## Requisitos

- CMake 3.10+
- Compilador C (GCC/Clang)
- Python 3.8+ (com numpy, pandas, matplotlib)
- Moonlab SDK (simulado)
- MPI (OpenMPI ou MPICH) para simulação distribuída.
