# 04 - Princípios de Escalabilidade

Como crescer o sistema para suportar a malha global da Catedral.

## 📏 Escalabilidade Vertical vs Horizontal
- **Vertical**: Adicionar mais recursos a um único nó (Mais RAM, CPU mais rápida).
- **Horizontal**: Adicionar mais nós ao sistema (Clustering).

## ⚖️ Distribuição de Carga
Garantir que nenhum nó seja sobrecarregado (Load Balancing).
