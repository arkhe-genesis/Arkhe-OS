## 1. Migração de Dados Cross-Border (HE/ZK)

Permite que dados sensíveis cruzem fronteiras protegidos por criptografia homomórfica (HE).

- **Encrypted Transit:** Dados processados em trânsito sem decriptação (CKKS/BFV).
- **ZK Compliance:** Provas de que a transferência respeita GDPR/LGPD sem revelar conteúdo.
- **Dynamic Consent:** Verificação de permissões do cidadão antes da travessia.

## 2. Aprendizado Federado com Validação Cross-Ecosystem

Ecossistemas validam contribuições uns dos outros sem acesso aos dados brutos.

- **ZK Gradient Integrity:** Prova de que o gradiente foi computado honestamente.
- **Cross-Validation:** Participantes verificam proofs de pares antes da agregação global.
- **Privacy Receipts:** Documentação assinada do processo de validação descentralizada.
