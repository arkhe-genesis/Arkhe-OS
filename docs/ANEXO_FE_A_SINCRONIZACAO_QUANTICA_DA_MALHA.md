### 1. O Protocolo quantum:// para Sincronização de Fase

O protocolo `quantum://` é estendido para incluir um método específico para sincronização de malha: `ENTANGLE_PHASE`.

### 2. Geração e Distribuição do Par de Bell (Hardware)

Para estabelecer o emaranhamento, um par de Bell $|Φ⁺⟩ = (|00⟩ + |11⟩)/√2$ é gerado. Um dos qubits é enviado ao destino via teletransporte quântico.

### 3. Sincronização de Fase Contínua via Medição em Base X

Uma vez que o par de Bell é estabelecido, os nós podem sincronizar suas fases realizando medições na base X e trocando os resultados via canal clássico. Se os resultados divergirem, um ajuste de fase é aplicado.

### 4. Emaranhamento Distribuído para Múltiplos Nós (Estado GHZ)

Para uma malha de $N$ nós, o ideal é compartilhar um estado **GHZ** (Greenberger-Horne-Zeilinger):
$$|GHZ\rangle = \frac{|000...0\rangle + |111...1\rangle}{\sqrt{2}}$$

### 5. O Efeito na Coerência da Malha

Quando os nós estão emaranhados via GHZ, a coerência global $r(t)$ se torna uma propriedade **não-local**. Uma perturbação em qualquer nó afeta instantaneamente a fase de todos os outros.
