## 1. Arquitetura SIEM (Security Information and Event Management)

O Vigia ingere logs de todos os substratos (Rede, Identidade, Dados, Enclaves) e aplica regras de correlação KQL (Katedral Query Language):

- **Injestão Multi-Cloud:** Conectores nativos para AWS CloudTrail, Azure Activity Log e GCP Audit Logs.
- **Detecção de Anomalias:** Modelos de ML que detectam desvios de comportamento em DIDs e acessos a chaves KMS.
- **Threat Intelligence:** Integração com feeds globais para identificar vetores de ataque conhecidos em tempo real.

---

## 2. Orquestração SOAR (Security Orchestration, Automation, and Response)

Ao detectar uma ameaça, o Vigia ativa playbooks automatizados sem intervenção humana:

- **Isolamento de Workload:** Migração imediata de enclaves suspeitos para redes de quarentena.
- **Revogação de Credenciais:** Invalidação atômica de tokens OIDC comprometidos via mTLS.
- **Rotação de Emergência:** Disparo de rotação de Master Keys em HSMs distribuídos se um vazamento for detectado.
