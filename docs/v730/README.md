# Arkhe v7.3.0 — Quantum Frontiers Integration

This release brings six quantum‑enabled dimensions into the Cathedral.

- **Quantum Cloud API** – unified access to IBM, IonQ, Rigetti via standard REST/GraphQL.
- **Edge Computing** – AWS Greengrass / Azure IoT Edge with low‑latency Phi‑C sync.
- **TPU Accelerator** – JAX/XLA backend for high‑dimensional quantum simulations.
- **ARKHE IMMERSIVE** – AR/VR visualization of quantum states (WebXR).
- **Satellite PXE** – remote deployment over Starlink/OneWeb DTN.
- **QPU Firmware** – direct pulse scheduling and calibration controller.

See `config/v730/` for deployment manifests and `docs/` for API references.
