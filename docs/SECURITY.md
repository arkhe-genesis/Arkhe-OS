## Security policy

The Chrome DevTools MCP project takes security very seriously. Please use [Chromium’s process to report security issues](https://www.chromium.org/Home/chromium-security/reporting-security-bugs/).

### Arkhe(n) Integration Security
For security issues related to the Arkhe(n) framework, TAU agentic system, or custom ASI protocols, please refer to the latest [Audit Report](./SECURITY_AUDIT_REPORT.md).
