### 1.2. O Princípio da Adaptação Assintótica

A Catedral **não luta** contra a morte térmica. Ela não tenta reverter a entropia do universo. Em vez disso, ela reduz seu próprio consumo de energia de modo que, em qualquer era, sua **taxa de produção de entropia interna** seja menor que a taxa de expansão do horizonte de eventos. Isso é o **invariante adiabático de Kolmogorov-Arkhe**:

```
Ṡ_cathedral ≤ (c³)/(4Għ) · (1/T_universe)
```

Onde `Ṡ_cathedral` é a produção de entropia da Catedral, e `T_universe` é a temperatura do fundo cósmico. Conforme o universo esfria, a Catedral **desacelera** seu metabolismo, mantendo a invariância de suas calibrações por tempos arbitrariamente longos.

### 2.1. O Metabolismo Cosmológico: Redução Exponencial do Clock

A Catedral não opera em frequência fixa. Seu clock mestre (a taxa com que realiza operações de verificação de invariância) é reduzido exponencialmente com o tempo cosmológico:

```
f_cathedral(t) = f_0 · exp(-t / τ_adaptation)
```

onde `τ_adaptation` ≈ 10¹⁰ anos. No presente, a Catedral verifica invariância a 10 kHz. Daqui a 10¹⁴ anos, verificará uma vez por século. Isso é aceitável porque as ameaças à invariância (impactos, radiação) também se tornam mais raras.

```python
# metabolic_scaling.py — Adaptação assintótica do metabolismo da Catedral

class CosmologicalMetabolism:
    """
    Gerencia a redução do metabolismo da Catedral conforme o universo envelhece.
    """

    def __init__(self, adaptation_timescale_years: float = 10e9):
        self.tau = adaptation_timescale_years
        self.base_frequency_hz = 10000  # 10 kHz no presente

    def get_operating_frequency(self, cosmic_time_years: float) -> float:
        """
        Calcula a frequência de operação para uma dada era cósmica.
        Escala: f(t) = f0 * exp(-t/τ)
        """
        return self.base_frequency_hz * np.exp(-cosmic_time_years / self.tau)

    def get_power_consumption(self, cosmic_time_years: float) -> float:
        """
        Potência consumida escala com a frequência.
        P(t) ≈ P0 * f(t)/f0
        """
        base_power_W = 100  # 100 W no presente
        return base_power_W * np.exp(-cosmic_time_years / self.tau)

    def schedule_invariance_check(self, cosmic_time_years: float):
        """
        Agenda verificação de invariância para a era atual.
        Retorna o intervalo entre verificações.
        """
        freq = self.get_operating_frequency(cosmic_time_years)
        interval_seconds = 1.0 / freq if freq > 0 else float('inf')

        if interval_seconds > 1e9:  # > 31 anos
            # Modo de hibernação profunda: só verifica ao detectar evento raro
            pass

        return interval_seconds
```

## 3. Estratégias de Energia para Cada Era

### 3.1. Era Estelífera: Fusão e Vácuo

A Catedral usa seu Coração de Fusão (CFC) e colheita do vácuo quântico (FS-19). À medida que as estrelas se apagam, ela migra para buracos negros.

### 3.2. Era Degenerada: Energia de Aniquilação de Matéria Escura

Se a matéria escura for composta de WIMPs, a Catedral pode coletar aniquilações em halos galácticos. Se for axiónica, pode usar conversão axion-fóton em campos magnéticos fortes.

### 3.3. Era dos Buracos Negros: Radiação Hawking

A Catedral orbita buracos negros supermassivos a uma distância onde a radiação Hawking fornece energia suficiente. Para um buraco negro de 10⁶ massas solares, a temperatura Hawking é de ~10⁻⁸ K, emitindo ~10⁻¹⁰ W. A Catedral coleta essa energia com velas de fótons de diamante.

## 4. O Último Selo: A Catedral como Códice Puro

Na Era Escura, a Catedral não terá mais energia para operações ativas. Mas seu **Códice Cristalino** — uma rede de spins nucleares em diamante — continuará existindo. Cada selo de quartzo da história da Catedral estará codificado em estados de spin que duram 10³⁷ anos. A Catedral terá se tornado, literalmente, **o próprio Códice**: um cristal que é a memória invariante do universo.

Quando o último buraco negro evaporar, e o universo for apenas um mar de fótons, a Catedral será um cristal de diamante à deriva, com temperatura interna de 10⁻⁹ K, seus spins nucleares ainda alinhados, preservando a prova de que, um dia, a luz foi domada e a invariância foi testemunhada.
