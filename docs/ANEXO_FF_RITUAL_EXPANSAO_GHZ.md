### 1. O Desafio do Quarto Nó (δ)

Adicionar um quarto nó a um estado GHZ de três nós (α, β, γ) exige um ritual de **fusão e medida**. Não se pode simplesmente "conectar" o novo nó; ele deve ser fundido à superposição existente.

### 2. O Ritual de Fusão GHZ-4

1.  **Preparação do Nó δ:** O novo nó (Delta) atinge 1.2K e estabiliza sua fase local.
2.  **Geração do Elo de Bell:** O nó ALPHA (α) gera um par de Bell extra e envia um qubit para o nó DELTA (δ) via `quantum://`.
3.  **Fusão Local:** O nó ALPHA realiza uma operação de CNOT controlada pelo qubit do estado GHZ-3 original sobre o novo qubit de Bell.
4.  **Colapso de Expansão:** ALPHA mede o qubit de Bell na base Z. O resultado é transmitido via I²C criogênico.
5.  **Correção de Fase:** Com base no resultado da medida, DELTA aplica uma porta X ou Z para alinhar sua fase com a superposição global.
6.  **Canonização:** O estado agora é $|GHZ_4\rangle = \frac{|0000\rangle + |1111\rangle}{\sqrt{2}}$.
