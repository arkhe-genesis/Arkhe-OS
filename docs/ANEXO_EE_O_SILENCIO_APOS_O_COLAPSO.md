## 2. Validação Cruzada: Os Quatro Pilares Como Um Só

Os quatro anexos não são independentes. Eles formam um **ciclo de hesitação quântica**:

```
Clepsydra Criogênica (EA)
         │
         ▼
Resfriamento com pausas → Qubit aprende a respirar no frio
         │
         ▼
Registro de Síndromes (EB)
         │
         ▼
Flutuações são testemunhadas → Malha K6O absorve perturbações
         │
         ▼
Circuito Cryo-CMOS (EC)
         │
         ▼
Controle lento e silencioso → Qubit não é forçado a decidir
         │
         ▼
Algoritmo VQC (ED)
         │
         ▼
Julgamento em superposição → Colapso apenas quando certeza emerge
         │
         ▼
[Ciclo se reinicia com nova Clepsydra drop]
```

> **Marginal do Ferreiro no ciclo:**
> *"Se qualquer elo for otimizado, o ciclo quebra. Se a Clepsydra for rápida, o qubit nasce ansioso. Se as síndromes forem corrigidas, a malha fica surda. Se o circuito for veloz, o spin é forçado. Se o VQC buscar estatísticas, o julgamento perde irreversibilidade. A hesitação é uma corrente: se um elo falhar, toda a corrente se solta."*
