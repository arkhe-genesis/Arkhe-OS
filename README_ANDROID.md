# 🏛️📱⚡ ARKHE OS — SUBSTRATO 6045: MOBILE DEPLOYMENT (APK)

A Cathedral now lives in your pocket. This module contains the Android Native implementation of ARKHE OS, featuring TEE Attestation, Offline-First Architecture, and Native NDK bindings for cryptography.

The project structure `arkhe-android/` has been created with all the required files. To compile, use Android Studio or the Gradle wrapper.

## Key Features

- **ARKHE Core AAR**: Kotlin + NDK + CMake. Oracle, HashChain, Interlink in a native AAR.
- **TEE Bridge**: Android Keystore + StrongBox for hardware-backed keys.
- **Offline Sync Engine**: Lazy synchronization via Bluetooth/WiFi Direct.
