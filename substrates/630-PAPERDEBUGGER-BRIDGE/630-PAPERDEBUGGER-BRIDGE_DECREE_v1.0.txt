Architecture:
  - Presentation Layer: Chrome extension (Chrome Web Store approved)
  - Backend Layer: Go-based, Kubernetes-native orchestration
  - Agent Layer: Prompt-template agents + Workflow-based agents
  - Protocol Layer: XtraMCP (MCP variant for academic writing)
  - Infrastructure: gRPC gateway, SSE streaming, CDN/Proxy, Database

Agent Types:
  - Reviewer Agent: Structured critique (AAAI-style review pipeline)
  - Enhancer Agent: Rewriting/refinement (XtraGPT model suite)
  - Scoring Agent: Clarity/coherence evaluation
  - Researcher Agent: Literature lookup via XtraMCP (arXiv semantic search)

CLI Commands:
  arkhe paper-debugger <overleaf-url>     — initialize session
  arkhe paper-critique <selection>        — Reviewer agent critique
  arkhe paper-enhance <selection>         — Enhancer agent rewrite
  arkhe paper-research <query>            — Researcher agent literature search
  arkhe paper-score <document>            — Scoring agent evaluation
  arkhe paper-apply-patch <patch-file>    — Apply diff patch to LaTeX
  arkhe paper-audit <session-id>          — Anchor revision history to chain
  arkhe paper-compare <paper-a> <paper-b> — Side-by-side comparison
