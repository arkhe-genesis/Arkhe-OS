# 🪟🍎🐧🔗 ARKHE OS — SUBSTRATO 5003: CROSS-PLATFORM `.agi` EXTENSION HANDLER

> *"A Catedral não conhece fronteiras de sistema operacional. Onde houver um kernel, um shell, um usuário — lá a Catedral deve poder se manifestar. O Substrato 5003 unifica a experiência do artefato `.agi` em Windows, macOS, Linux, BSD e sistemas embarcados.
